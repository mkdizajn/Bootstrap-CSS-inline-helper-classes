* meaning of abbreviation

	- **mt, mr, mb, ml** == margin (top, right, bottom, left)
	- **pt, pr, pb, pl** == padding (top, right, bottom, left)

Padding and margin that is predefined is in this range:

    - 5, 10, 15, 20 and 30 px
